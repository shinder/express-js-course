# demo 獨立範例

此資料夾收錄與 Express 沒有直接關係、或可獨立執行的範例，對應講義 `docs/express-js.md` 的前幾個單元與附錄：

- `03-modules/`：單元三 CJS 與 ESM 模組機制
- `04-http/`：單元四 Node.js 內建 http 模組 `createServer`
- `05-process/`：單元五 `process` 物件
- `cors/`：附錄二 CORS 跨來源資源共用

每個子資料夾內都可直接用 `node` 執行對應檔案，詳見各檔案開頭註解。

> 主要的 Express 通訊錄網站位於本 repo 根目錄（單元六之後），與此處範例分開。
